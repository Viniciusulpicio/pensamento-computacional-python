Este repositório contém os exercícios, notas de aula e projetos práticos desenvolvidos durante a disciplina de **Pensamento Computacional**, como parte do Bacharelado em IA.

O objetivo é documentar o aprendizado da lógica de programação e a aplicação dos quatro pilares do pensamento computacional: **Decomposição, Reconhecimento de Padrões, Abstração e Algoritmos.**

## Objetivos do Repositório
* Praticar a sintaxe da linguagem **Python**.
* Desenvolver o raciocínio lógico para a resolução de problemas complexos.
* Estruturar algoritmos eficientes e legíveis.
* Servir como base de consulta para fundamentos de programação.

## Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Ambiente :** [Google Colab (Executar Notebook Online)](https://colab.research.google.com/github/Viniciusulpicio/Pensamento-Computacional-Python/blob/main/PensamentoComputacionalIA_A.ipynb)

* `PensamentoComputacionalIA_A.ipynb`: Notebook principal com a implementação da lógica de programação, estruturas de dados e exercícios práticos da disciplina.
